Take Action

You can start reducing waste today by:
  • Using the reusable takeaway container when leaving the dining hall with food.
  • Using a reusable water bottle instead of buying more plastic.
  • Going digital! Avoid printing paper when you can.
  • Only buy the food you need. If you have extra meal swipes or dining dirhams, consider asking if someone needs them! (or, let it go)
